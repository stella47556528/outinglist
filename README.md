# outinglist
📖 行程小幫手 (Planner) 使用說明書

歡迎使用「行程小幫手」！這是一款結合文青美學與實用功能的行動化行程管理工具。本程式採用離線優先設計，並具備自動資料清理功能，讓您的計畫清單永遠保持簡潔。

🎨 核心視覺指南

莫蘭迪橘紅 (Terracotta)：代表「進行中」或「今天」的任務，提醒您保持專注。

鼠尾草綠 (Sage Green)：代表「已達成」的目標，給予完成任務後的平和感。

石灰色 (Stone)：系統介面與導覽色調，營造文青質感。

🛠 功能操作說明

1. 新增行程

點擊標頭右側的「＋」按鈕開啟視窗。

日期選擇：系統鎖定不可選擇今天以前的日期。

儲存並繼續：儲存目前行程後不關閉視窗，自動清空欄位以便連續輸入下一筆。

儲存並結束：儲存後關閉視窗並回到列表。

2. 列表管理與過濾

狀態標籤：分為「進行中」、「全部清單」、「已達成」。

日期過濾：可切換「今天」、「明天」、「未來行程」與「已過期」。

快速跳轉：點擊頂部中央的「今天進行中」計數器，系統會一鍵篩選出今天所有待辦行程。

3. 行程卡片操作

完成請按這：點擊後行程會變色並移至已達成。

修改行程：點擊卡片頂部的「鉛筆」圖示，可調整日期、地點與細節。

刪除行程：點擊卡片頂部的「垃圾桶」圖示，行程將移入回收紀錄。

4. 回收紀錄 (Archive)

點擊右下角的深灰色垃圾桶按鈕開啟面板。

恢復：點擊旋轉箭頭圖示將行程放回主清單。

永久刪除：點擊叉叉圖示徹底移除紀錄。

⚖️ 系統維護規則

⏳ 三天自動清理機制

為了保持清單的整潔與效能，系統具備以下自動化邏輯：

僅保留過去三天：系統啟動時，會自動檢查並永久刪除日期早於今天三日以上的所有紀錄（包含回收站）。

過期提醒：過期行程（未滿三日）會標示為「已過期」並以灰色文字呈現，提醒您補辦或歸檔。

💾 資料存檔

程式會自動將所有內容儲存在您的瀏覽器本地空間 (LocalStorage)。

關閉程式或重新整理頁面，資料均不會消失。

📱 PWA 行動裝置安裝

您可以點擊畫面下方的「安裝」按鈕，或透過瀏覽器選單選擇「加入主畫面」。

安裝後可像原生 App 一樣全螢幕操作，且在無網路環境下仍可正常開啟與紀錄。

自動儲存：所有資料會自動儲存在您的手機瀏覽器中，重新開啟也不會消失。

離線啟動：即使在斷網環境下，App 仍可正常載入並讀取您已儲存的內容。
