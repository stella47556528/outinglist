# outinglist
這趟出門要做的事
📝 行程小幫手 (Planner) 使用說明書歡迎使用「行程小幫手」！這是一款專為行動裝置優化的行程與待辦紀錄 App。以下是各項功能的詳細操作指南：1. 核心功能新增行程：點擊頂部右側的「＋」按鈕，會彈出輸入視窗。輸入「要去哪裡」後點擊加入即可。待辦項目提醒：畫面頂部正中央會顯示剩餘的待辦數量。點擊該區域可快速跳轉至「進行中」清單。標記完成：點擊行程卡片下方的翡翠綠按鈕「完成後請按這」，該項目會變色並標示為完成。2. 管理與修改修改內容：點擊行程卡片右上角時間旁邊的「鉛筆」圖示進入編輯模式。貼心設計：若原本顯示為「未註明事項」，進入編輯時會自動清空文字，方便直接輸入。未儲存警示：在編輯模式下，若未按下儲存就點擊畫面其他地方，系統會跳出彈窗詢問是否要儲存變更。刪除行程：點擊卡片右側的「垃圾桶」圖示，項目會被移至回收站。3. 分類過濾您可以透過畫面上方的標籤切換顯示內容：進行中：顯示尚未完成的探險計畫。全部：顯示所有（包含已完成）的計畫。已完成：顯示已經達成的目標。4. 回收站（垃圾桶）機制點擊右下角的黑色垃圾桶按鈕可開啟回收站面板：恢復：點擊「旋轉箭頭」圖示，將行程放回主清單。永久刪除：點擊「垃圾桶」圖示，徹底移除該筆紀錄。5. PWA 特性與離線功能加入主畫面：本程式為 PWA（Progressive Web App）。在手機瀏覽器點擊「分享」後選擇「加入主畫面」，即可像原生 App 一樣從桌面啟動。自動儲存：所有資料會自動儲存在您的手機瀏覽器中，重新開啟也不會消失。離線啟動：即使在斷網環境下，App 仍可正常載入並讀取您已儲存的內容。祝您每一次出門都順利愉快！
